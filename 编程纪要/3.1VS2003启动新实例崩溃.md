## 3.1 VS2003启动新实例崩溃
###1.问题描述：
安装vs2003和vs2010后，vs2003启动新实例崩溃了，不能进行调试项目。
###2.原因分析：
安装vs2003和vs2010后，IIS默认情况下使用的是ASP.NET2.0的版本，而vs2003需要1.1版本，所以只要进入IIS->网站－>右键属性－>ASP.NET->下拉列表框中把版本选为上面的1.1就OK了。
###3.解决方案：
开始菜单里找到Microsoft Visual Studio .NET 2003->Visual Studio .NET 工具->Visual Studio .NET 2003 命令提示，在打开的窗口里运行了一下“aspnet_regiis /i”，重新注册一下。（如果到这一步还是不行则继续）再去IIS中的Web服务扩展里看，除了原来的ASP.NET v2.0.50727一项外终于又多出来了一项ASP.NET v1.14322，将其设置为允许。
